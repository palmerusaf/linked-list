# Linked-list

## Description
 A JS implementation of a linked list data structure with the following methods:
* append
* prepend
* size
* head
* tail
* at
* pop
* contains
* find
* toString

## Testing 
If you want to see the tests in action run the following commands:
```bash
git clone https://github.com/palmerusaf/linked-list.git
```
```bash
cd ./linked-list
```
```bash
npm i
```
```bash
npm test
```
